跨店对比

     - 周期显示:（缺省）当前日期所在周（周一至周日）
     - 日期筛选器:（1）选择任意天，选择范围截止至当天（2）缺省为当天
     - 跨店对比条件:筛选器日期范围内当前门店（店仓A)与店仓B销售额对比
  • 店仓B默认为时间范围内同办事处同品牌，除当前门店外 排名最高门店；
  • 显示各门店排名次序；
  • 销售金额＝单位时间内，成交价*销售数量（计入退货商品）；
  • 本店环比公式
    •      （本时间周期销售金额－同年上一周期销售金额）／本期金额＊100%
  • 本店同比公式
    •      （本时间周期销售金额－去年上一周期销售金额）／本期金额＊100%
  • 同环比为正数时标 ↑；负数时标 ↓；

购物篮分析

     - 日期筛选范围内，每天的
  • 日小票数量（ ＊ 计入退单数量）；
  • 搭配率
    •      日销售商品数量／日小票数量 （＊ 不计入退货商品）
  • 客单价
    •      日均销售总额／日小票数量；
  • 件单价

活动分析（饼图，环比统计）

  • 对比日期
    •      当前日期范围与同年上期
  • 外环数值
    •      日期范围内所有促销活动销售额／销售总额＊100%（＊正价＝促销活动的一种）
  • 内环
    •      正价活动与所有促销活动销售总额相对比例；
